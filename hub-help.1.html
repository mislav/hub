Show the help page for a command.

Synopsis

hub help hub
hub help COMMAND
hub help hub-COMMAND [--plain-text]

Options

hub-COMMAND
Use this format to view help for hub extensions to an existing git command.
--plain-text
Skip man page lookup mechanism and display raw help text.

hub(1), git-help(1)
